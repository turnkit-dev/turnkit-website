Getting Started

Unity Quickstart

1. Download TurnKit.unitypackage and import into your Unity project.

2. Fill project name and continue to login. After login everything is generated and linked.

Get Started Window

3. Use Clone Project & Open New Editor, or for more control use the Unity menu at TurnKit/ParrelSync.

4. Run the sample scene in two Unity editors.

Note: The client key is only shown on first creation. If you need a new one later, create it in Dashboard.
